# 📊 Sales Performance Analysis Dashboard

## Project Overview

This project analyzes sales transactions across different regions, cities, products, sales channels, and sales representatives. The goal is to uncover key business insights, identify top-performing products and regions, and evaluate overall profitability.

The analysis was conducted using Microsoft Excel, including data cleaning, transformation, pivot tables, and dashboard creation.

---

## Dataset Information

The dataset contains sales records with the following fields:

* Date
* Region
* City
* Customer Type
* Channel
* Product
* Category
* Unit Price
* Quantity
* Revenue
* Cost of Goods Sold (COGS)
* Profit
* Sales Representative
* Cost Price
* Month

---

## Objectives

The project aims to:

* Analyze total sales performance
* Evaluate profit generation across products and regions
* Identify top-performing products
* Compare sales channels
* Monitor monthly sales trends
* Assess sales representative performance
* Support data-driven business decisions
